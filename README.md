🚀 Основные возможности
📝 Управление заметками
➕ Создание, редактирование и удаление заметок
📌 Закрепление важных заметок вверху списка
🔍 Поиск и фильтрация по всем заметкам
⏰ Напоминания к заметкам (по времени)
✨ Редактор и Markdown
Поддержка Markdown-разметки с предпросмотром
Разделение на две панели: редактор / результат
Черновики кода и временные списки задач прямо в приложении
Оформление (ThemeManager)
12 цветовых тем на выбор:
Светлые, тёмные, высококонтрастные, пастельные и монохромные
У каждой темы свои цвета для:
Фона (bg), текста (fg), акцентов (accent)
Кнопок (button, button_text) и состояния наведения (hover)
🧩 Интерфейс
Область	Что делает
Верхняя панель	Заголовок, строка поиска, кнопки действий (создать, темы, экспорт/импорт)
Боковая панель	Список всех заметок, закреплённые сверху, контекстное меню (ПКМ)
Центральная зона	Редактор + предпросмотр Markdown (раздельно или рядом)
Нижняя панель	Статусная строка: кол-во заметок, последнее автосохранение, напоминания
🧠 Зачем заметки в разработке? (Идея проекта)
«Заметки в IDE или десктопном блокноте — это не просто список дел»
Проект решает реальные задачи программиста и не только:
Фиксация идей «на лету»
Работаете над проектом, а в голову пришло решение для другого? Открыли NotesApp, записали — не переключая контекст.
Черновики кода
Набросали прототип функции, проверили логику и только потом перенесли в основной код.
Временные списки задач
«Поправить документацию», «протестировать функцию X», «написать тесты» — всё под рукой.
Быстрая навигация по заметкам
Закрепление важного + поиск позволяют не теряться, даже если заметок > 100.
Снижение когнитивной нагрузки
Выгружаете мысли, задачи и сниппеты в NotesApp — освобождаете мозг для текущей разработки.
🛠 Технологии
Язык: Python 3.10+
Графический фреймворк: Tkinter (стандартная библиотека)
Markdown-парсер: markdown (или misaka — на ваш выбор)
Работа с архивами: zipfile (стандартная библиотека)
JSON: json (стандартная библиотека)
Управление временем: threading, datetime
