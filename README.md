Это полноценное десктопное приложение-блокнот с богатым функционалом:
Создание, редактирование, удаление заметок
Поддержка Markdown-разметки
12 цветовых тем оформления
Экспорт/импорт в MD, JSON, ZIP
Напоминания к заметкам
Автосохранение
Поиск и фильтрация
Закрепление важных заметок
ThemeManager         Управление 12 цветовыми темами
DataManager          Загрузка/сохранение заметок и настроек в JSON
ExportImportManager  Экспорт/импорт в разные форматы, резервное копирование
SplashAnimation      Анимация загрузки при запуске
NotesApp             Главный класс приложения (UI и логика)
Хранит 12 предустановленных тем. Каждая тема — это словарь с цветами:
bg — фоновый цвет
fg — цвет текста
accent — акцентный цвет (для выделений)
button, button_text — цвета кнопок
hover — цвет при наведении
Интерфейс состоит из:
Верхняя панель — заголовок, поиск, кнопки действий
Боковая панель — список всех заметок
Центральная область — редактор с предпросмотром
Нижняя панель — статусная строка
